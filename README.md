# Minha Solução — Banco

## Stack
- **Backend:** Python 3.12, FastAPI Standard v0.138.2
- **Frontend:** React + Vue

## Pré-requisitos / dependências
- Poetry para as dependências de backend
- NPM para as dependências do frontend

## Como executar

### Backend (API)
```bash
poetry run fastapi dev
```

Após o procedimento, você pode explorar a api em http://localhost:8000/docs

### Frontend
```bash
npm install
npm run dev
```

Após isso, acesse http://localhost:5173/ para começar a usar a interface
